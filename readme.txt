This program implements the PSI-PDE method for discovering PDEs from dynamical systems.
The code PSI_algorithm.m conducts sparse regression using the PSI-algorithm in the PSI-PDE method.
The codes solve*.m solve and optimize the learned PDEs.

Please refer to the following GitHub repository for the codes for numerial methods in the folder \Gcode including 
1. differx.m
2. differy.m
3. differ3.m
4. differfitx.m
5. coeff.m
https://github.com/Cao-WenBo/PDE_identification
